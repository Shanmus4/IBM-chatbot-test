#### FIFA Refereeing Platform (2025)
**Project Overview:**
Designed a comprehensive enterprise system for referee management at FIFA, serving stakeholders across 211 member associations and 6 confederations worldwide.

**Challenge:**
The existing referee nomination and approval process relied heavily on manual emails, spreadsheets, and scattered external tools, leading to missed deadlines, duplicate data, and time-consuming tracking for thousands of stakeholders.

**My Role:**
- User Research
- UX Design
- Interaction Design
- Design System Development

**Process:**
- Conducted weekly calls with FIFA administrators
- Interviewed 14 different member associations and 5 confederations
- Analyzed existing workflows and pain points
- Customized Google's Material Design 3 system to align with FIFA guidelines
- Built flexible, reusable component library
- Shipped design iteratively over 8 months

**Key Features Designed:**
- **Admin Dashboard:** Unified control panel for managing nominations, approvals, and configurations
- **Nomination Management:** Bulk action capabilities, virtual scrolling, duplicate detection system
- **Approval Tracking:** Collapsible confederation views, real-time status monitoring
- **Configuration Panel:** Email template management, deadline settings, multi-language support
- **Association Microsite:** Three-step nomination process (select, rank, submit)
- **Confederation Portal:** Grid and list views for referee vetting and approval

**Design Decisions:**
- Action bar with checkboxes for bulk operations (admins work on multiple items simultaneously)
- Four-level navigation using side navigation instead of top menus
- Virtual scrolling instead of pagination (only 211 associations to manage)
- Segmented buttons for related content views
- Horizontal scroll tables with customizable column order
- Read-only and edit modes for configuration settings
- Automated duplicate detection and management

**Impact:**
- **3x increase** in association nomination speed
- **1.5x acceleration** in confederation approval speed
- **62% increase** in nomination submission rate in first window
- **181 associations** submitted on time (up from 112 previous season)
- **100% elimination** of duplicate entries (saving 300+ manual corrections per season)
- **4.2/5 average satisfaction score** from associations and confederations
- **5/5 rating** from primary FIFA administrator

**Awards:**
- Winner, Infosys Insta Awards 2025

**Team:**
- Shanmu (Designer)
- Mandy Grimmiger (Design Lead)
- Stefanie Borel (Design Manager)

**Note:** Project protected by NDA. Details shared reflect publicly available information.
